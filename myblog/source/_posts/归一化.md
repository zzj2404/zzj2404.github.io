# MinMaxScaler

$$
\frac{data - min}{max-min}
$$


```python
from sklearn.preprocessing import MinMaxScaler

# 归一化到 [-1,1]
scaler = MinMaxScaler(feature_range=(-1, 1))

# 对于在一个列表中的单维度值，改变形状为多样本单维度
values = values.reshape(-1,1)

values = scaler.fit_transform(values)

```
